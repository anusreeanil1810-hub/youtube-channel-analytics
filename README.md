# YouTube Channel Analytics

## 📌 Project Overview

YouTube Channel Analytics is a web-based application developed to analyze and display statistics of YouTube channels. The application uses **ShaktiDB** as the backend database to efficiently store and retrieve channel information, making it easy to view analytics in a user-friendly interface.

## ✨ Features

* View YouTube channel details
* Display subscriber count
* Display total views
* Display total uploaded videos
* Store and retrieve analytics using ShaktiDB
* Simple and user-friendly interface

## 🛠️ Technologies Used

* **Programming Language:** Python
* **Backend Database:** ShaktiDB
* **Operating System:** Ubuntu
* **Version Control:** Git
* **Repository Hosting:** GitHub

## 🚀 Installation

1. Clone the repository:

```bash
git clone https://github.com/anusreeanil1810-hub/youtube-channel-analytics.git
```

2. Navigate to the project directory:

```bash
cd youtube-channel-analytics
```

3. Create a virtual environment:

```bash
python3 -m venv venv
```

4. Activate the virtual environment:

```bash
source venv/bin/activate
```

5. Install the required packages:

```bash
pip install -r requirements.txt
```

6. Run the application:

```bash
python app.py
```

## 📊 How It Works

1. The user enters or selects a YouTube channel.
2. The application processes the requested data.
3. ShaktiDB stores and manages the analytics data.
4. The application displays the channel statistics in an organized format.
